Particle Filter

E09 Advanced Topics Particle Methods Pendulum + Friction Est. 1993

When noise is non-Gaussian or the system has discontinuities, particle filters represent the full posterior distribution using a weighted set of samples.

Overview

The Kalman filter and EKF assume Gaussian noise and (approximately) linear dynamics. Many real systems violate these assumptions: Coulomb friction creates a discontinuity at $\dot\theta=0$, contact forces produce impulsive non-Gaussian disturbances, and uncertain initial conditions can make the posterior distribution bimodal.

The particle filter (also called Sequential Monte Carlo) represents the posterior $p(\mathbf{x}_k|\mathbf{y}_{1:k})$ as a weighted set of $N$ samples (particles). It makes no assumptions about the shape of the distribution and can track multimodal, skewed, or heavy-tailed posteriors.

Key Insight

The particle filter is the only standard estimator that can faithfully represent a bimodal posterior. When a pendulum might be swinging left or right, particles naturally split into two clusters — something no Gaussian filter can do.

When to Use Particle Filters

  • Non-Gaussian noise — Uniform, Cauchy, mixture distributions
  • Multimodal posteriors — Uncertain initial direction, ambiguous measurements
  • Discontinuous dynamics — Coulomb friction, contact, switching modes
  • Highly nonlinear systems — Where EKF linearization fails

Key Properties

  • Asymptotically exact — Converges to true posterior as $N\to\infty$
  • No linearity assumption — Works with arbitrary nonlinear models
  • Computationally expensive — Cost scales linearly with $N$
  • Weight degeneracy — Requires resampling to prevent collapse

How It Works

The Bootstrap Particle Filter (SIR — Sampling Importance Resampling) operates in three steps each time a measurement arrives:

  1. Predict (Propagate): Draw each particle forward through the nonlinear dynamics model, including process noise: $$\mathbf{x}_k^{(i)} \sim p(\mathbf{x}_k | \mathbf{x}_{k-1}^{(i)})$$ For our pendulum this means integrating $\ddot\theta = -(g/L)\sin\theta - \mu\,\mathrm{sign}(\dot\theta) - d\dot\theta + w$ for each particle.
  2. Update (Weight): Evaluate the measurement likelihood for each particle and update its weight: $$w_k^{(i)} \propto w_{k-1}^{(i)} \cdot p(\mathbf{y}_k | \mathbf{x}_k^{(i)})$$ Particles close to the measurement get high weight; those far away get low weight.
  3. Resample: When the effective sample size $N_\mathrm{eff}$ drops below $N/2$, resample: duplicate high-weight particles and discard low-weight ones using systematic resampling (low-variance). Reset all weights to $1/N$.
Resampling Trade-off

Resampling combats weight degeneracy but introduces sample impoverishment — diversity is lost as particles are duplicated. The $N_\mathrm{eff}$ threshold balances these two effects. Watch the $N_\mathrm{eff}$ plot to see when resampling events occur.

Key Equations

Bayesian Recursion

$$p(\mathbf{x}_k | \mathbf{y}_{1:k}) \propto p(\mathbf{y}_k | \mathbf{x}_k) \int p(\mathbf{x}_k | \mathbf{x}_{k-1})\, p(\mathbf{x}_{k-1} | \mathbf{y}_{1:k-1})\, d\mathbf{x}_{k-1}$$

Particle Approximation of the Posterior

$$p(\mathbf{x}_k | \mathbf{y}_{1:k}) \approx \sum_{i=1}^{N} w_k^{(i)}\, \delta(\mathbf{x}_k - \mathbf{x}_k^{(i)}), \qquad \sum_{i=1}^{N} w_k^{(i)} = 1$$

Importance Weight Update

$$\tilde{w}_k^{(i)} = w_{k-1}^{(i)} \cdot p(\mathbf{y}_k | \mathbf{x}_k^{(i)}), \qquad w_k^{(i)} = \frac{\tilde{w}_k^{(i)}}{\sum_{j=1}^{N}\tilde{w}_k^{(j)}}$$

Gaussian Measurement Likelihood

$$p(y_k | \mathbf{x}_k^{(i)}) = \frac{1}{\sqrt{2\pi R}} \exp\!\left(-\frac{(y_k - h(\mathbf{x}_k^{(i)}))^2}{2R}\right)$$

Effective Sample Size

$$N_\mathrm{eff} = \frac{1}{\sum_{i=1}^{N} (w_k^{(i)})^2}$$

When $N_\mathrm{eff} \ll N$, most weight is concentrated on a few particles and resampling is needed. The range is $1 \le N_\mathrm{eff} \le N$.

Systematic Resampling

Generate one uniform draw $u_0 \sim \mathcal{U}[0, 1/N)$ and set $u_i = u_0 + i/N$. Walk through the cumulative weight distribution to select indices. This gives $O(N)$ resampling with lower variance than multinomial resampling.

Pendulum + Coulomb Friction Dynamics

$$\ddot\theta = -\frac{g}{L}\sin\theta - \mu\,\mathrm{sign}(\dot\theta) - d\,\dot\theta + w, \qquad w \sim \mathcal{N}(0, Q)$$

The $\mathrm{sign}(\dot\theta)$ term is discontinuous at $\dot\theta=0$, making the process noise distribution non-Gaussian through the dynamics.

MATLAB Implementation

MATLAB
%% E09: Bootstrap Particle Filter (Pendulum + Coulomb Friction)
function xnext = pendulumDynamics(x, dt, mu)
    g = 9.81; L = 1.0; d = 0.05;
    theta = x(1); thetaDot = x(2);
    thetaDDot = -(g/L)*sin(theta) ...
                - mu*sign(thetaDot) - d*thetaDot;
    xnext = x + dt*[thetaDot; thetaDDot];
end

%% Systematic resampling
function idx = systematicResample(w, N)
    csw = cumsum(w);
    csw = csw / csw(end);       % normalise
    u0 = rand/N;
    u = u0 + (0:N-1)'/N;
    idx = zeros(N,1);
    j = 1;
    for i = 1:N
        while csw(j) < u(i), j = j+1; end
        idx(i) = j;
    end
end

%% Main PF loop
N_particles = 200; dt = 0.02; T = 5;
mu = 0.8; Q = 0.01; R = 0.1;
Nsteps = round(T/dt);

% True state
x_true = [0.8; 2.0];

% Bimodal prior: half particles +vel, half -vel
particles = zeros(2, N_particles);
for i = 1:N_particles
    dir = 2*(i <= N_particles/2) - 1;
    particles(:,i) = [0.8+0.3*randn; dir*2+0.5*randn];
end
w = ones(1,N_particles) / N_particles;

for k = 1:Nsteps
    % Measurement
    y = x_true(1) + sqrt(R)*randn;

    % Predict: propagate particles
    for i = 1:N_particles
        particles(:,i) = pendulumDynamics(particles(:,i), dt, mu);
        particles(:,i) = particles(:,i) + ...
            sqrt(Q)*[0.1*randn; randn]*dt;
    end

    % Update: weight by likelihood
    for i = 1:N_particles
        w(i) = w(i) * normpdf(y, particles(1,i), sqrt(R));
    end
    w = w / sum(w);

    % Effective sample size
    N_eff = 1 / sum(w.^2);

    % Resample if N_eff < N/2
    if N_eff < N_particles/2
        idx = systematicResample(w, N_particles);
        particles = particles(:, idx);
        w = ones(1,N_particles) / N_particles;
    end

    % Weighted mean estimate
    x_est = particles * w';

    % Propagate true state
    x_true = pendulumDynamics(x_true, dt, mu);
    x_true = x_true + sqrt(Q)*[0.1*randn; randn]*dt;
end
